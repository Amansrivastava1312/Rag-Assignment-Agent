ANSWER:
Object-oriented programming (OOP) is a method of organizing code using classes and objects to model real components and manage complexity.

### Key Concepts
*   **Class:** Defines the data and behavior.
*   **Object:** A real instance of a class.
*   **Design Principles:** Key principles include single responsibility, open-closed, dependency inversion, low coupling, and high cohesion. Specifically, a class should have one main reason to change (Single Responsibility).

### Why use OOP?
OOP is useful for large projects because it groups related logic, improves reuse, and makes systems easier to extend. It should be used when state and behavior belong together.

```python
class Pipeline:
    pass
```

### Important Considerations
*   **Common Mistake:** Do not use OOP just to look advanced. If a function is sufficient, use a function.
*   **Use Case:** Use OOP when you need to model real components within a system where state and behavior are inherently linked.
